coinbase-tracker
======
A minimal console ticker for Coinbase exchange rates, written in Python 3.3.


Screenshot
------------
![Screenshot](http://i.imgur.com/ooXJ5nD.png)


### Usage

        $ ./tracker.py [btc amount]
